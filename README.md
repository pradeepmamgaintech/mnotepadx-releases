# MNotepadX Releases

Official Windows releases for **MNotepadX**.

MNotepadX is a lightweight desktop text editor designed for a simple and focused writing experience.

## Official Website

https://mamgain.app/apps/mnotepadx

## Download

Download the latest version from the **Releases** section:

https://github.com/pradeepmamgaintech/mnotepadx-releases/releases/latest

Each release may include:

* **Setup version** — recommended for most users

## System Requirements

* Windows 10 or later
* 64-bit Windows

## Installation

### Setup Version

1. Download the latest `MNotepadX-Setup-<version>-x64.exe` file.
2. Run the installer.
3. Follow the on-screen instructions.

## Security Notice

Windows may display a security warning because the application is not yet code-signed. Confirm that you downloaded the file from this repository or the official MNotepadX website before continuing.

## About This Repository

This repository is used only for distributing official MNotepadX releases.

The application source code is maintained privately.
